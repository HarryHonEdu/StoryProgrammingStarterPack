Story's Game Development Starter Pack

Hi! This is a project I created called Story's Gaming Development Starter Pack.

Want to learn game development but don't know how or where to start? Then, this project is perfect for you! In this project, I've gathered tons of useful resources that will help you eventually start developing your own game!

Throughout this starter pack, you'll learn the basics of game development, including key programming languages like C#, C++, and Java, which are widely used in game engines such as Unity and Unreal Engine. You'll also get familiar with game design principles, development workflows, and how to use essential documentation and resources that will guide you in your game creation journey.

By the end, you'll have the foundational knowledge to dive deeper into more advanced topics and start building your own projects!
